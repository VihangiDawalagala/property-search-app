A client-side single-page web application built using React and Vite, inspired by Rightmove.

## Features
- Search properties by type, price, bedrooms, date added, and postcode
- View property details with image gallery and tabs
- Add and remove favourites (button + drag and drop)
- Responsive design for desktop and mobile
- Client-side security using CSP
- Unit testing with Jest

## Technologies Used
- React (Vite)
- Tailwind CSS
- Radix UI
- Jest & Testing Library
